## Requirement
* React
* Vite
* Firebase
* Tiptap

## Usage
Clone the repository

```angular2html
git clone https://github.com/dashluu/AirNotes-web.git
```

Go to the project directory and run

```angular2html
npm run dev
```

Open http://localhost:5173 to see the web app!
